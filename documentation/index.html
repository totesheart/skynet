Installation

Server Requirements :

  • PHP >= 7.3.0
  • BCMath PHP Extension
  • Ctype PHP Extension
  • Fileinfo PHP extension
  • JSON PHP Extension
  • Mbstring PHP Extension
  • OpenSSL PHP Extension
  • PDO PHP Extension
  • Tokenizer PHP Extension
  • XML PHP Extension

After downloading file from codecanoyn you will get a zip file extract it. You will see Three folder 1.beardbrand_cms, 2.multipurpose_website_cms 3. Portfolio_CMS.

Open folder which one you want to install. you will get a file name file.zip upload it on your Cpanel or localhost, and extract it.


now visit your domain or site URL. you will see an installer follow all steps. After complete those steps, your site is ready.
